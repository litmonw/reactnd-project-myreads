# 图书跟踪应用

## 运行应用

1. 打开项目文件，进入命令行，输入 npm install 安装所有依赖
2. 安装后，在命令行输入 npm start 运行项目
3. 图书图片使用了 Google Api，可能无法正常访问

## 技术栈

1. React + Router
2. npm
3. Books API
